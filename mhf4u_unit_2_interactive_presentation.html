The Division Statement

Once you finish dividing, you write the result as an equation.

P(x) = D(x) × Q(x) + R(x)

Dividend = (Divisor × Quotient) + Remainder

Just like saying: 13 = (5 × 2) + 3

Always insert zero for any missing degree in the dividend!

The Remainder Theorem

What if we ONLY care about the remainder?

Running a full division is a waste of time.

When a polynomial P(x) is divided by (x - k), the remainder is exactly equal to...

P(k)

Method 2: The Theorem

Evaluate P(2)

P(2) = (2)3 - 4(2)2 + 5(2) - 1

P(2) = 8 - 16 + 10 - 1

P(2) = 1

The Factor Theorem

If 10 ÷ 5 has a remainder of 0, then 5 is a perfect factor.

The exact same logic applies to polynomials.

If P(k) = 0

then (x - k) is a Factor

Finding the First Root

P(x) = x3 - 7x - 6

Test factors of the constant (-6): ±1, ±2, ±3, ±6

P(1)

(1)3 - 7(1) - 6 = -12

Not a factor

P(-1)

(-1)3 - 7(-1) - 6 = 0

Bingo! (x + 1) is a factor.
